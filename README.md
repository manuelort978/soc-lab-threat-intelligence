# SOC Lab – Threat Intelligence Detection

## Overview

This project simulates the detection of malicious IP addresses by comparing web server logs against a threat intelligence list.

---

## Objectives

* Detect malicious IPs in logs
* Simulate threat intelligence integration
* Perform real-time monitoring

---

## Detection Logic

* Extract IPs from logs
* Compare against known malicious IP list
* Trigger alert on match

---

## Simulation

Traffic was generated from a known IP included in the malicious list.

---

## Features

* Real-time detection
* Simple threat intelligence integration
* Log correlation

---

## Skills Demonstrated

* Threat intelligence usage
* Log analysis
* Python scripting
* SOC detection workflows
